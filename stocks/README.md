# 股票研究資料目錄

此目錄包含所有進行深度研究的股票資料。

## 目錄結構

每個股票代碼都有獨立的目錄，結構如下：

```
[股票代碼]/
├── README.md          # 該股票的說明和研究歷史
├── research/          # 研究報告目錄
│   ├── v001_YYYY-MM-DD.md
│   ├── v002_YYYY-MM-DD.md
│   └── ...
└── summary.md         # 自動生成的對比摘要
```

## 如何新增股票研究

1. 在 `stocks/` 目錄下創建股票代碼目錄（如 `2330`、`AAPL`）
2. 在該目錄下創建 `research/` 子目錄
3. 將研究報告放入 `research/` 目錄，命名格式：`v[版本號]_[日期].md`
4. 更新根目錄的 `index.md` 索引文件
5. 更新該股票的 `README.md` 文件
6. 執行 `scripts/generate_summary.py` 生成對比摘要

## 檔案命名規範

- **研究報告**：`v001_2024-01-15.md`（版本號三位數，日期格式 YYYY-MM-DD）
- **股票目錄**：使用股票代碼（如 `2330`、`AAPL`、`TSLA`）
