# V-sign

股票深度研究資料庫 - 使用 Gemini Deep Research 進行定期股票分析

## 專案簡介

本專案使用 Gemini Deep Research 對多支個股進行每兩週一次的深度研究，包含行業分析、定價預測、風險評估等內容。所有研究報告均儲存在此 GitHub 儲存庫中，方便追蹤時間序下的預測變化。

## 目錄結構

```
V-sign/
├── README.md                    # 本文件
├── index.md                     # 股票索引（所有股票清單、最新研究狀態）
├── stocks/                      # 所有股票研究資料
│   ├── [股票代碼]/              # 例如：2330、AAPL
│   │   ├── README.md            # 該股票的說明和研究歷史
│   │   ├── research/            # 研究報告目錄
│   │   │   ├── v001_2024-01-15.md
│   │   │   ├── v002_2024-01-29.md
│   │   │   └── ...
│   │   └── summary.md           # 自動生成的對比摘要
│   └── TEMPLATE_README.md       # 股票 README 模板
│   └── TEMPLATE_REPORT.md       # 研究報告模板
└── scripts/
    └── generate_summary.py      # 自動生成對比摘要的腳本
```

## 快速開始

### 查看所有股票

瀏覽 [index.md](index.md) 查看所有進行研究的股票清單及其最新研究狀態。

### 查看特定股票

1. 進入 `stocks/` 目錄
2. 選擇股票代碼目錄（如 `2330`、`AAPL`）
3. 查看該股票的 `README.md` 了解研究歷史
4. 查看 `research/` 目錄中的各版本報告
5. 查看 `summary.md` 了解時間序對比分析

### 範例：查看台積電（2330）的研究

```
stocks/2330/
├── README.md          # 研究歷史和基本資訊
├── research/
│   ├── v001_2024-01-15.md
│   ├── v002_2024-01-29.md
│   └── v003_2024-02-12.md
└── summary.md         # 價格預測變化、觀點演變對比
```

## 使用指南

### 新增股票研究

當您完成一次新的 Gemini Deep Research 研究後，請按照以下步驟提交：

#### 1. 準備研究報告

使用 `stocks/TEMPLATE_REPORT.md` 作為模板，確保報告包含以下章節：
- 執行摘要
- 行業分析
- 定價預測
- 風險評估
- 競爭分析
- 其他分析

#### 2. 建立股票目錄（首次研究時）

```bash
# 假設要新增台積電（2330）的研究
mkdir -p stocks/2330/research
```

#### 3. 複製並編輯股票 README

```bash
# 複製模板
cp stocks/TEMPLATE_README.md stocks/2330/README.md
# 編輯 README.md，填入股票基本資訊
```

#### 4. 儲存研究報告

將研究報告儲存到 `research/` 目錄，命名格式：`v[版本號]_[日期].md`

```bash
# 例如：第一份報告
stocks/2330/research/v001_2024-01-15.md
# 第二份報告（兩週後）
stocks/2330/research/v002_2024-01-29.md
```

**命名規範**：
- 版本號：三位數，從 `001` 開始遞增
- 日期格式：`YYYY-MM-DD`
- 範例：`v001_2024-01-15.md`、`v002_2024-01-29.md`

#### 5. 更新索引

編輯 `index.md`，在股票清單表格中新增或更新該股票的資訊：
- 股票代碼
- 股票名稱
- 市場
- 最新版本號
- 最新研究日期
- 研究次數
- 連結到股票目錄

#### 6. 更新股票 README

編輯該股票的 `README.md`：
- 更新研究歷史表格
- 更新最新研究摘要
- 確保所有報告連結正確

#### 7. 生成對比摘要

執行自動化腳本生成 `summary.md`：

```bash
python scripts/generate_summary.py [股票代碼]
# 例如：
python scripts/generate_summary.py 2330
```

腳本會自動：
- 解析所有研究報告
- 提取價格預測資訊
- 提取關鍵觀點
- 生成對比表格和趨勢分析

### 研究報告格式要求

為確保自動化腳本能正確解析，請遵循以下格式：

#### 定價預測章節格式

```markdown
## 定價預測

### 目標價格分析
- **目標價格**：600 元
- **當前價格**：580 元
- **潛在漲跌幅**：3.45%
- **預測時間範圍**：6個月

### 價格預測依據
[詳細說明...]
```

#### 執行摘要格式

關鍵觀點請使用列表格式：

```markdown
## 執行摘要

### 核心結論
- [結論 1]
- [結論 2]
- [結論 3]
```

#### 風險評估格式

```markdown
## 風險評估

### 風險等級評估
- **整體風險等級**：中
- **風險評分**：6/10
```

## 工具與腳本

### generate_summary.py

自動生成股票研究對比摘要的 Python 腳本。

**使用方法**：
```bash
python scripts/generate_summary.py [股票代碼]
```

**功能**：
- 掃描指定股票的所有研究報告
- 提取價格預測、關鍵觀點、風險評估
- 生成對比表格和趨勢分析
- 輸出到 `stocks/[股票代碼]/summary.md`

**依賴**：
- Python 3.6+
- 無需額外套件（僅使用標準庫）
